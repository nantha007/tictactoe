# Tic tac toe using Q learning

## Overview

The project uses Q learning to implement AI for the tic tac toe. 

## Run instructions

Clone the repository using the follwing commands.
```
git clone --recursive /home/nuke07/ARC/Tictactoe/tictactoe.py
```

To run from the terminal, enter the following commands.

```
python3 tictactoe.py
```
